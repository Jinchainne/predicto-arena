## Title

Predicto Arena - GenLayer Prediction Market Terminal

## Description

Predicto Arena is a GenLayer-powered prediction market terminal. Users browse live-style markets, filter categories, open market detail layers, inspect outcome prices, view order book and rules tabs, simulate trades, review portfolio positions, and track leaderboard/ticket missions.

The project is GenLayer-native because market settlement needs transparent oracle resolution. GenLayer can read public sources, evaluate ambiguous event outcomes, and support AI-assisted consensus for market rules and outcome verification.
